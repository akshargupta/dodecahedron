# dodecahedron

Assembly tracker for 30-unit modular origami dodecahedra with a proper 3-edge coloring. Live at [dodecahedron.vercel.app](https://dodecahedron.vercel.app).

Built with [Vite](https://vitejs.dev/), vanilla JavaScript, and SVG.

## Develop

```bash
npm install
npm run dev      # localhost:5173
npm run build    # build to dist/
npm run preview  # preview the build
```
